# Crop Detection System

AI-based crop disease detection using ESP32 and Python server.

## Features
- Crop disease prediction
- ESP32 hardware integration
- Telegram bot alerts
- Python backend server

## Technologies Used
- Python
- ESP32
- Arduino IDE
- Machine Learning

## Project Structure
- AURDINO CODE FOR ESP32
- PYTHON SERVER

## How to Run
1. Upload ESP32 code
2. Run Python server
3. Connect hardware
4. Send image/data for prediction
